execute: 메서드는 모든 종류의 SQL문을 실행할 수 있는 메서드
INSERT, UPDATE, DELETE, SELECT 등을 자동으로 인식하지는 않음
값을 BOOLEAN 값으로 반환
TRUE는 ResultSet(데이터를 반환하는 Select 문)에 대한 결과를 나타냄
false는 Update 문(INSERT) 도는 다를 유형의 sql문을 반환

executeQuery: 데이터를 조회하고 반환하기 위해 SQL SELECT문을 실행하는데 주로 사용
			  실행 결과를 나타내는 ResultSet 객체를 반환

executeUpdate: 이 메서드는 주로 DB에서 영향을 미치는 SQL문 (INSERT, UPDATE, DELETE) 
			   실행하는데 사용
			   SQL문으로 인해 영향을 받은 행의 수를 반환
			   데이터 변경 작업의 성공 여부를 나타내는 정보
